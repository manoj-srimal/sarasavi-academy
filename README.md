# Sarasavi English Academy - Official Website 🎓

A modern, highly responsive, and beautifully animated web application built for **Sarasavi English Academy**. This platform serves as the digital front door for the academy, showcasing their courses, student memories, and providing a seamless way for new students to register.

## 🚀 Live Demo
https://aquamarine-lolly-c8b4f1.netlify.app/

## 🛠️ Tech Stack & Technologies
* **Framework:** Next.js (React)
* **Styling:** Tailwind CSS
* **Animations:** Framer Motion
* **Icons:** Lucide React
* **Deployment:** Netlify

## ✨ Key Features
* **Premium UI/UX:** A custom grey-white gradient theme providing a highly professional and academic look.
* **Smooth Animations:** Integrated Framer Motion for scroll-triggered animations, hover effects, and page transitions.
* **Direct Application Download:** One-click PDF application download embedded directly into the navigation flow.
* **Fully Responsive:** Optimized for Mobile, Tablet, and Desktop views.
* **Quick Contact Integrations:** Direct links to WhatsApp, email, and Google Maps.
